# Analog-Circuits-Fault-Diagnosis-Based-on-Machine-Learning
dataset and project fille
data folder: save the txt file of the original S parameters and the data file saved to EXCEL
txt2excel.py: The S parameters obtained through testing are saved in a txt file, and the txt file is saved to an EXCEL file in a specific format.
PCA.py: Read EXCEL file data, reduce the dimensionality of the data and display graphics
SVM.py: SVM multi-classifier, evaluate model performance
DT.py: Decision tree multi-classifier, evaluate model performance
